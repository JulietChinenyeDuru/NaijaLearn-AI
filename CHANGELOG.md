## [1.0.0]

### First Release

#### Added

- Multilingual STEM and ICT tutoring in 5 Nigerian languages
  - English
  - Pidgin English
  - Yoruba
  - Hausa
  - Igbo

- 10 core subjects
  - ICT and Computer Science
  - Programming — Python, Java, C++, Web Development
  - Data Science and Machine Learning
  - Mathematics and Statistics
  - Physics
  - Chemistry
  - Biology
  - Engineering — Electrical, Mechanical, Civil
  - Economics and Accounting
  - Research Methodology and Academic Writing

- 5 academic levels
  - 100 level (Year 1)
  - 200 level (Year 2)
  - 300 level (Year 3)
  - 400 level (Final year)
  - Postgraduate

- University selector covering 250+ NUC accredited Nigerian universities

- Interactive quiz mode — one question at a time with instant feedback

- Smart subject suggestions — quick question prompts per subject

- Conversation memory — tutor remembers context within a session

- Fully responsive mobile layout — optimised for Nigerian students on Android phones

- Google Analytics GA4 integration — tracks student usage by subject, language and university

- UptimeRobot monitoring — 5-minute uptime checks with email alerts

- Netlify serverless function backend — API key securely hidden server-side

- Abia State University Uturu campus photo in hero section

- Secure Claude Haiku API integration via Netlify Functions
